```sh
pip install numpy==1.10.1
```

## Vurtual Env
Каждый раз при работе в проекте нужно создавать виртуальное окружение, чтобы пакеты не путались. Это важно.

Создать окружение в текущем каталоге:

```
python -m venv env
```

Активировать его:

```sh
source env/bin/activate
```

Деактивировать так:

```sh
deactivate
```
